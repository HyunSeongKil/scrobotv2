상상했던
나만의 홈페이지를
만들어 보세요

코딩을 못해도 멋진 홈페이지가 되고,
당신의 아이디어를 자유롭게 펼칠 수 있는 방법

무료로 시작하기 →

SC_ROBOT 에디터를 사용하면

코딩 없이 쉽고 자유롭게 웹 서비스를 할 수 있어요!

  • 다양한 디자인 요소

    에디터에서 제공하는 다양하고
    강력한 기능의 디자인 요소들을
    사용해 보세요.

  • 드래그 ＆ 드롭

    마우스로 다양한 요소들을 이동하며
    쉽고 간편하게 홈페이지를 완성하세요.

  • 알아서 자동으로

    화면만 그리세요!
    DB 조작, 배포, 코딩 같은 복잡한 일은
    SC_ROBOT이 알아서 할게요.

SC_ROBOT 에디터는

다양한 기능이 있어요!

  • 자동 데이터베이스 처리

    원하는 기능을 지정하면 자동으로
    DB테이블 생성부터 SQL문까지
    작성해드려요

  • 클릭 한 번의 편리한 배포

    클릭 한 번만으로 내가 그린 화면이
    호스팅이 되고 바로 서비스가
    가능해요

  • 팝업창, 화면이동 링크

    그린 화면을 팝업창 지정 또는
    화면이동 지정으로 쉽고 자유롭게
    선택할 수 있어요

  • 고품질 코드 자동 생성

    화면을 그리면 표준화된
    고품질 소스코드를
    자동으로 생성해드려요

  • 편리한 스타일 지정

    나만의 개성 있는 컴포넌트를 디자인
    할 수 있고 CSS코드를 입력할
    필요 없이 마우스 클릭으로
    마음껏 바꿔보세요

  • 계단식 메뉴 관리

    한 눈에 볼 수 있는 계단식 메뉴구조로
    쉽게 메뉴를 편집하고
    자유롭게 관리할 수 있어요

SC_ROBOT 에디터는

안전하고 빨라요!

  • 보안

    SC_ROBOT은 철저한 보안을 유지해요.
    시큐어 코딩은 물론 데이터를 항상 암호화해서 전송하기
    때문에 안심하고 사용하실 수 있어요.

  • 데이터 관리

    SC_ROBOT은 백업 기능을 제공해요.
    매 시간마다 데이터베이스와 각종 파일들을 백업해서 회원님의
    소중한 데이터를 최대한 복원할 수 있도록 노력하고 있어요.

  • 속도

    SC_ROBOT은 서버 속도를 최고로 유지하고 있어요.
    항상 최고의 속도를 유지해 회원님이 불편하지 않도록
    노력하고 있어요.

  • 유연한 서버

    홈페이지에 생각보다 많은 사람들이 방문하게 되면 서버가
    동작하지 못하게 되요. SC_ROBOT은 클라우드를 사용해서
    홈페이지가 잘 동작하도록 서버를 쉽게 늘려 유연하게 서버
    문제를 관리할 수 있어요.
